# Spell_Card_Converter
A programm to convert the csv from 5eTools to a json which the Spellbook Card Generator can interpret.
Usage:
Go to https://5e.tools/spells.html and select the Spells you want to print by clicking on them (multi selection possible) and pinning them to your list.
Then go to the table view of that list and download the SCV.
Pull the current state of this project and install all neccessary dependencies.
Run main, choose the file you just downloaded and click convert.
It generates the xml in the same folder as the source file.
Open the xml in https://beta.hardcodex.ru/spells.
Done.
